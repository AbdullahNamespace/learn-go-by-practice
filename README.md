# 🐹 Learn Go by Practice

## 📖 About | حول المشروع

A collection of Go programming exercises for learning through practice. Each problem is documented with clear descriptions in both English and Arabic.

مجموعة من تمارين برمجة Go للتعلم من خلال الممارسة. كل مشكلة موثقة بأوصاف واضحة بالإنجليزية والعربية.

---

## 📂 Repository Structure | هيكل المستودع

```
learn-go-by-practice/
├── .gitignore                 # Git ignore file | ملف تجاهل الجيت
├── README.md                  # Project overview | نظرة عامة على المشروع
├── template.md                # Problem template | قالب صياغة المشاكل

    ├── 1_problem_name/

    │   └── main.go
    └── ...
```

---

## 🚀 Getting Started | البدء

### Prerequisites | المتطلبات

- Go installed ([Download](https://go.dev/dl/))

```bash
go version
```

### Running a Problem | تشغيل مشكلة

```bash
# Clone the repository | استنساخ المستودع
git clone https://github.com/AbdullahNamespace/learn-go-by-practice.git
cd learn-go-by-practice

# Run a specific problem | تشغيل مشكلة محددة
go run problems/1_problem_name/main.go
```
